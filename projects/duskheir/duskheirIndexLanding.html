DuskHeir

Combat-focused 3D action game developed in Godot 4.6. The project revolves around a global combat-state mechanic based on SUN, MOON and DUSK combat alignments.

Godot 4.6 GDScript Forward+ Jolt Physics Architecture Driven
OVERVIEW

Project Overview

DuskHeir is built around a single core mechanic: dynamic state switching between SUN, MOON and DUSK.

Every major gameplay system reacts to the current state:

  • Combat interactions
  • Abilities
  • Enemy behaviors
  • Damage systems
  • Visual feedback
  • Resource management
